Features

Everything included

Stop wiring together separate tools. OxideTerm covers terminal, files, editing, forwarding, and AI — and they share state so you never context-switch.

Local & remote terminal

Local PTY (zsh, bash, fish, pwsh, WSL2) and SSH in one window. Split panes, broadcast keystrokes, WebGL renderer with adaptive 3-tier refresh. 100 000-line scroll buffer.

Smart reconnect

When SSH drops, a 30-second grace period probes the old connection. If it recovers, TUI apps (vim, htop, yazi) keep running. On full reconnect, forwards, transfers, and IDE state are restored.

SFTP & local file manager

Dual-pane browser with drag-and-drop, batch rename, and real-time transfer queue. Rich previews: images, video, audio, PDF, hex, and 30+ syntax languages.

Remote IDE

CodeMirror 6 over SFTP — no server installation needed. Lazy file tree with Git badges, project-wide search, multi-tab with LRU eviction, pin tabs, and optimistic-lock conflict resolution.

Port forwarding

Local (-L), remote (-R), and dynamic SOCKS5 (-D). All managed through the UI, auto-restored on reconnect. Lock-free I/O: channel owned by a single task.

30+ themes & visual editor

Dracula, Nord, Catppuccin, Tokyo Night, and more. Edit all 22 xterm.js colour fields and 19 app-chrome CSS variables. Background image gallery with per-tab opacity and blur.

Architecture

Under the hood

Technical decisions that make a real difference at runtime — in the source, not just marketing copy.

Pure Rust SSH — no OpenSSL

russh 0.54 with ring crypto backend. Zero C dependencies in the SSH path. Supports ChaCha20-Poly1305, AES-GCM, Ed25519, RSA, ECDSA, OpenSSH certificates, keyboard-interactive 2FA, and a custom AgentSigner for native SSH agents on macOS, Linux, and Windows.

8-stage reconnect pipeline

queued → snapshot → grace-period → ssh-connect → await-terminal → restore-forwards → resume-transfers → restore-ide. Grace period probes the old connection for 30 s before destructive reconnect. If the link recovers, TUI apps never see a SIGHUP. All logic flows through the orchestrator store — nothing reconnects directly from event handlers.

Dual-plane binary protocol

Terminal I/O travels over WebSocket: [Type:1B][Length:4B BE][Payload:nB]. Four frame types: Data, Resize, Heartbeat, Error. Zero JSON on the hot path. Control commands (SFTP, settings, connections) use Tauri IPC on a separate plane. Reference-counted SSH pool shares one physical connection across terminal, SFTP, and port forwards.

Security

Credentials never touch config files

Everything sensitive goes through the OS keychain or is encrypted before leaving memory.

OS keychain — always

SSH passwords under com.oxideterm.ssh, AI keys under com.oxideterm.ai. Config files hold reference IDs only — never raw secrets. Touch ID for AI key access on macOS. Uses macOS Keychain, Windows Credential Manager, or Linux libsecret.

.oxide encrypted export

Portable backup format: ChaCha20-Poly1305 AEAD + Argon2id KDF (256 MB memory, 4 iterations). SHA-256 integrity check. Optional private key embedding. Pre-flight analysis flags missing keys before export.

Defence in depth

Host key TOFU via ~/.ssh/known_hosts — rejects changed fingerprints. Plugin sandbox with Object.freeze(), Proxy ACL, circuit breaker, and IPC whitelist. Sensitive buffers cleared via zeroize. Rust ownership prevents use-after-free at language level.
